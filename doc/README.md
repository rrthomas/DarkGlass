# DarkGlass

DarkGlass is a simple content management system that makes a static web site from a collection of files providing the content. The overall look of the site is controlled by HTML templates. DarkGlass pages can be written as HTML or Markdown.

DarkGlass comes with a simple web server that you can use to test a site in development.

Get the code, file issues &c. on [GitHub](https://github.com/rrthomas/DarkGlass).
